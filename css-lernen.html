Design-Mission

CSS macht deine Website schön.

CSS bestimmt Farben, Größen, Abstände, Kästen und Layouts. Ohne CSS ist eine Website nur Inhalt. Mit CSS bekommt sie Charakter.

1. Farbe

h1 { color: #00a6ff; }

Mit `color` änderst du die Textfarbe.

2. Kästen

.karte { padding: 20px; border: 2px solid blue; }

Mit Abstand und Rahmen baust du echte Bereiche.

3. Layout

.grid { display: grid; gap: 16px; }

Mit Grid kannst du Inhalte sauber nebeneinander anordnen.

Aufgabe 1

Welche CSS-Eigenschaft ändert die Textfarbe?

Wähle die richtige Antwort.

Tipp: Textfarbe heißt auf Englisch `color`.

Aufgabe 2

Welche Eigenschaft macht Innenabstand?

Innenabstand ist der Platz zwischen Inhalt und Rand einer Box.

Tipp: Innenabstand heißt `padding`.
