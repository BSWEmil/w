Kleine Anfänge

HTML lernen

HTML ist die Sprache für den Inhalt einer Website. Mit Tags wie Überschriften, Texten, Links und Buttons entsteht die Grundstruktur. Unten kannst du eigenen Code schreiben und sofort sehen, was passiert.

1. Überschrift

<h1>Meine Überschrift</h1>

Mit `h1` machst du die wichtigste Überschrift auf einer Seite.

2. Text

<p>Das ist ein Text.</p>

Mit `p` schreibst du normale Textabschnitte.

3. Link

<a href="index.html">Zur Startseite</a>

Mit `a` kannst du auf andere Seiten verlinken.

Aufgabe 1

Schreibe das fehlende HTML-Tag.

Welche Tags brauchst du, damit der Text eine große Überschrift wird?

< >Meine Website</ >

Tipp: Die wichtigste Überschrift heißt h1.

Aufgabe 2

Baue einen Link.

Welches HTML-Tag benutzt man, um auf eine andere Seite zu verlinken?

Tipp: Ein Link braucht `href`.
