Interaktions-Mission

JavaScript macht deine Website lebendig.

JavaScript reagiert auf Klicks, ändert Texte, zählt Punkte und kann Dinge anzeigen oder verstecken. Das ist der Teil, bei dem eine Seite anfängt zu reagieren.

1. Element finden

const text = document.getElementById("text");

So findest du ein HTML-Element über seine ID.

2. Text ändern

text.textContent = "Neuer Text";

Damit ersetzt du den Inhalt eines Elements.

3. Klick ausführen

<button onclick="start()">Start</button>

Beim Klick wird eine JavaScript-Funktion gestartet.
