Abschluss-Mission

Baue deine erste eigene Website.

Hier wird aus Lernen ein echtes Projekt. Wer diese Schritte schafft, hat die Grundlage für eine eigene Website verstanden.

01

Thema wählen

Überlege dir, worüber deine Website sein soll: über dich, ein Hobby oder ein Spiel.

02

HTML schreiben

Baue Überschrift, Texte, Links, Bilder-Platzhalter und Buttons ein.

03

CSS gestalten

Gib deiner Website Farben, Kästen, Abstände und ein klares Layout.

04

JavaScript testen

Füge eine kleine Aktion hinzu, zum Beispiel einen Button, der Text verändert.

Starter-Code

Grundgerüst

Damit kann jeder anfangen. Danach wird die Seite Stück für Stück persönlicher.

<!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <title>Meine Website</title> </head> <body> <h1>Meine erste Website</h1> <p>Hier schreibe ich meinen ersten Text.</p> </body> </html>

Projekt-Check

Plane deine eigene Seite direkt hier.

Fülle die Felder aus. Wenn alles ausgefüllt ist, bekommst du deine Projekt-Mission.

Tipp: Je genauer dein Plan, desto leichter wird der Code.

Extra

Teste deinen Code live.

Gehe zurück zum HTML-Testfeld und probiere dein eigenes Grundgerüst aus.
